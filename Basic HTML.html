The Importance Of Coding Everyday.

The importance of coding everyday is such that at least 20 minutes a day will allow you to keep growing in knowledge of coding languages and keep you from going backwards leading to procrastination and failure in your goals.

The Importance of Projects.

The importance of projects is of high value and learning. This high value of learning will involve running into difficulties that you can learn from such as hot to structure code or how to code in a way that solves an unforeseen problem. This unforeseen problem could be how to code a new way to deliver user messages or how to code a system to handle user posts more efficiently.

Value Of Tutorials

The value of tutorials is of medium to high importance with the use of some moderation. Moderation must be used as to not get caught in tutorial hell or in tutorial rut that seems you cant get out of which leads to possible burnout.
